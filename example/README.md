# Sleep Stage Chart Example

这是一个展示如何使用 `sleep_stage_chart` 插件的示例应用。

## 运行示例

1. 确保你已经安装了 Flutter SDK
2. 在项目根目录运行以下命令：

```bash
cd example
flutter pub get
flutter run
```

## 示例功能

- 展示完整的睡眠阶段图表
- 包含浅睡、深睡、快速眼动和清醒阶段
- 支持触摸交互和指示器显示
- 提供图例说明

## 代码说明

示例代码位于 `lib/main.dart` 文件中，展示了：

- 如何创建睡眠数据
- 如何配置图表参数
- 如何使用自定义画笔绘制图表
- 如何添加图例和说明
